VS 플러그인
---
```
1. Vetur
   - Vue.js 코드에 대한 문법 강조, 코드 자동완성, 디버깅, 린팅 기능들을 제공
2. Night Owl
   - 코드강조 플러그인
3. Material Icon Theme
   - Icon 테마
4. Live Server
   - html을 구동하기 위한 Server 환경 Plug-in
5. ESLint
   - 자바스크립트 문법 중 에러가 있는 곳에 표시를 달아놓는 도구
6. Prettier
   - 코드를 읽어들여서 사용자 옵션에 따라 코드를 다시 포맷팅하는 코드 포맷터
7. Auto Close Tag
   - 태그를 자동으로 닫아주는 Plug-in
8. Atom Keymap
   - 기본단축키 변경 툴
```
